# chatwebsocketio
 chat application, using websocketio and express. Inserted inside a docker container.
 
 
 
### dependencies 
node 14+ npm 7+

### How install
npm install

### How config 
name	default	type	required	description
PORT	3333	number	yes	the port of endpoint API
### How run
npm run start

### Port enable
define PORT in .env

### Endpoints
**Url** : `http://localhost:3333/`
